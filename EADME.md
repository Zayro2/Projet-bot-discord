# Discord IA Bot

Bot Discord qui répond comme une IA à chaque fois qu'on le mentionne.

## Commande
- Mentionne le bot dans un message, par exemple : `coucou @Bot`
- Le bot te répond grâce à l'API OpenAI

## Installation
1. Installer Node.js
2. Cloner le repo
3. Créer un fichier `.env` avec tes clés Discord et OpenAI
4. Installer les dépendances : `npm install`
5. Lancer le bot : `npm start`
